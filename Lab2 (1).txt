/* 1. Виділення першого елемента <li> та надання йому жирного шрифту */
li:first-child {
    font-weight: bold;
}

/* 2. Вибір усіх елементів <a>, які є останніми дочірніми елементами свого батька, і фарбування їх у червоний */
a:last-child {
    color: red;
}

/* 3. Вибір усіх елементів input, які вимкнено (disabled), і надання їм сірого фону */
input:disabled {
    background-color: gray;
}

/* 4. Вибір усіх елементів <p>, які є прямими дочірніми елементами <div>, і надання їм напівжирного шрифту */
div > p {
    font-weight: bold;
}

/* 5. Вибір усіх елементів, які мають атрибут href, що починається з https, і фарбування їх у зелений */
a[href^="https"] {
    color: green;
}

/* 6. Вибір усіх елементів input з атрибутом type="text" і обведення їх синьою рамкою */
input[type="text"] {
    border: 1px solid blue;
}

/* 7. Вибір усіх елементів <li>, які є першими або останніми дочірніми елементами */
li:first-child, li:last-child {
    font-size: 24px;
}

/* 8. Вибір усіх елементів <p>, які є нащадками елемента <div>, і встановлення для них розміру шрифту 16 пікселів */
div p {
    font-size: 16px;
}

/* 9. Вибір усіх елементів input зі значенням checked і надання їм світло-блакитного фону */
input:checked {
    background-color: lightblue;
}

/* 10. Виділення усіх елементів <a>, які є прямими дочірніми елементами <li>, і підкреслення їх тексту */
li > a {
    text-decoration: underline;
}

/* 11. Виділення всіх парних елементів <li> та надання їм жирного шрифту */
li:nth-child(even) {
    font-weight: bold;
}

/* 12. Вибір усіх елементів input з типом text, які не disabled, і надання їм сірого фону */
input[type="text"]:not(:disabled) {
    background-color: gray;
}

/* 13. Вибір усіх елементів <p>, які є першими дочірніми елементами, і встановлення для них розміру шрифту 24 пікселі */
p:first-child {
    font-size: 24px;
}

/* 14. Вибір елемента <li>, який йде після елемента з класом .special, і надання йому перекресленого тексту */
.special + li {
    text-decoration: line-through;
}

/* 15. Виділення першої літери кожного елемента <p> та надання їй розміру шрифту 36 пікселів */
p::first-letter {
    font-size: 36px;
}

/* 16. Додавання ::after для кожного елемента <a> та надання йому вмісту ↗ */
a::after {
    content: "↗";
}
